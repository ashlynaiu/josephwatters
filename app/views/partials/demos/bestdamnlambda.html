Best Damn Lambda Explaination

Simply put, lambda functions are anonymous functions. They can be used anywhere a function can be used. They can be assigned to variables.

When we create a lambda function without assignment or execution we are just creating a place in memory for that funciton.

>>> lambda x: x+1 at 0x104191c80>

A lambda function in the raw can be read "lambda inputParameter: expression to evaulate on inputparameter". When assigned to a variable that variable is then treated like reference to a function and can be utilized the same way as a function.


>>> f = lambda x: x+1
>>> f(1)
2

If you are looking to evaluate inline, without assignment, then the declaration of the lamba function and the input value are wrapped in parenthesis side-by-side.


>>> (lambda x: x+1)(1)
2

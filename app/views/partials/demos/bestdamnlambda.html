Best Damn Lambda Explaination

Simply put, lambda functions are anonymous functions. They can both be used anywhere a function can be used and can be assigned to a variable. That's it! Well, for an explaination. Usage is a little tricky. So let's take a look at how we can use lambda functions.

When we create a lambda function without assignment or execution we are just creating a place in memory for that funciton.

>>> lambda x: x + 1

A lambda function in contains all of the aspects of a regular function. If we assign our function to a variable

>>> f = lambda x: x+1
>>> f(1)
2

If you are looking to evaluate inline, without assignment, then the declaration of the lamba function and the input value are wrapped in parenthesis side-by-side.

>>> (lambda x: x+1)(1)
2
